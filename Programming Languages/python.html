    What is Python?

    Python is a popular programming language. It was created by Guido van Rossum, and released in 1991.

    • web development (server-side)
    • software development
    • mathematics
    • system scripting

    Basic Syntax

    Python's syntax is clean and easy to understand. Here is a simple example that prints "Hello, World!" to the console:

    print("Hello, World!")

    Variables and Data Types

    In Python, you don't need to declare the type of a variable explicitly. The type is inferred from the value assigned to it.

    # Integer
    x = 10
    print(type(x))  # Output: <class 'int'>
    
    # Float
    y = 10.5
    print(type(y))  # Output: <class 'float'>
    
    # String
    name = "Python"
    print(type(name))  # Output: <class 'str'>

    Lists

    Lists are one of the most versatile data structures in Python. They can store items of different types.

    fruits = ["apple", "banana", "cherry"]
    print(fruits)  # Output: ['apple', 'banana', 'cherry']
    
    # Adding an item to the list
    fruits.append("orange")
    print(fruits)  # Output: ['apple', 'banana', 'cherry', 'orange']

    Loops

    Loops are used to iterate over a sequence (like a list, tuple, dictionary, set, or string).

    # For loop
    for fruit in fruits:
        print(fruit)
